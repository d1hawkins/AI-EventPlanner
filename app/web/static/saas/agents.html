How to Use AI Agents
Getting Started
  1. Select an agent from the list on the left
  2. Type your message in the chat input
  3. Press Enter or click the Send button
  4. Wait for the agent to respond
Agent Types
  • Event Coordinator: Orchestrates the event planning process
  • Resource Planner: Plans and manages resources for your event
  • Financial Advisor: Handles budgeting and cost estimation
  • And more... depending on your subscription tier
Tips for Best Results
  • Be specific in your requests
  • Provide context about your event
  • Ask follow-up questions if needed
  • Use the "Attach to Event" feature to give agents access to event details
Subscription Tiers
  • Free: Access to Coordinator and Resource Planning agents
  • Professional: Additional access to Financial, Stakeholder, Marketing, and Project Management agents
  • Enterprise: Full access to all agents including Analytics and Compliance & Security
