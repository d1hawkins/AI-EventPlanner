AI Agent Onboarding

Welcome to the AI Agent Onboarding page. This page provides a guide to using our AI agents for event planning.

This page is for demonstration purposes. The actual onboarding experience is integrated into the agents page.

What are AI Agents?

Our AI agents are specialized assistants that help you plan and manage your events. Each agent has specific expertise in different aspects of event planning.

Event Coordinator

Orchestrates the event planning process and delegates tasks to specialized agents

Resource Planner

Plans and manages resources needed for your event

Financial Advisor

Handles budgeting, cost estimation, and financial planning

How to Use AI Agents

  1. Select an agent from the list on the agents page
  2. Type your message in the chat input
  3. Press Enter or click the Send button
  4. Wait for the agent to respond
  5. Continue the conversation to plan your event

Tips for Best Results

  • Be specific in your requests
  • Provide context about your event
  • Ask follow-up questions if needed
  • Use the "Attach to Event" feature to give agents access to event details

I need to plan a corporate conference for 200 people in September.
