Analytics Dashboard Help

Using Filters

  • Date Range: Select start and end dates to view analytics for a specific time period
  • Agent Type: Filter data to show analytics for a specific agent type
  • Apply Filters: Click to update the dashboard with your selected filters
  • Reset: Clear all filters and show data for all time periods and agent types

Understanding Charts

  • Conversations Over Time: Shows the number of conversations started each day
  • Conversations by Agent: Shows the distribution of conversations across different agent types
  • Agent Usage Details: Provides detailed statistics for each agent type
  • Usage Trends: Shows your current usage against your subscription limits

Export Options

  • Export as CSV: Download analytics data as a CSV file for use in spreadsheet applications
  • Export as PDF: Download analytics data as a PDF report
  • Print Report: Print the current analytics dashboard

Tips for Analysis

  • Compare usage patterns over different time periods
  • Identify which agents are most popular with your team
  • Monitor your usage to ensure you stay within subscription limits
  • Use the detailed table to see which agents have the most engagement
